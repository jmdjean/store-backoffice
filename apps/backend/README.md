# Backend - Store Backoffice

API Node.js (Express) para o Store Backoffice.

## Desenvolvimento

```bash
npm run start:dev
```

API em `http://localhost:3000`.

## Endpoints

- `GET /health` - Health check
- `GET /api` - Info da API
